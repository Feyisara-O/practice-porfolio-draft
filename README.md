## About

A clean, responsive personal portfolio showcasing my projects and skills as a frontend developer. This was my very first portfolio — later refined and redesigned as I grew, but preserved here as the starting point.

## Features

- Sticky navigation header
- Hero section with profile photo
- Projects grid with live links
- About section with tech stack badges
- WhatsApp and phone contact links
- Fully responsive layout

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Markup | HTML5 |
| Styling | CSS3 (Custom Properties, Flexbox, Grid) |
| Fonts | DM Sans, Geist Mono (Google Fonts) |
| Hosting | GitHub Pages |

## What I Learned

- CSS custom properties (variables) for consistent theming  
- Flexbox and CSS Grid for responsive layouts  
- The `clamp()` function for fluid typography  
- `sticky` positioning for the navigation header  
- Mobile-first responsive design with media queries  
- Structuring a portfolio to communicate value to potential clients
